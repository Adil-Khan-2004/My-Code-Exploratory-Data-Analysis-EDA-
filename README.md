# Amazon Product Review Dataset - Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on the **Amazon Product Review Dataset** using Python. The goal is to explore, clean, and visualize the dataset to uncover meaningful insights about product ratings, reviews, pricing, discounts, and customer feedback.

## 🎯 Objectives

* Understand the structure of the dataset.
* Clean and preprocess the data.
* Handle missing values and duplicate records.
* Explore product ratings and review patterns.
* Analyze discounts, prices, and customer engagement.
* Visualize relationships between different features.
* Generate insights to support data-driven decision making.

## 🛠️ Technologies Used

* Python
* Jupyter Notebook / VS Code
* Pandas
* NumPy
* Matplotlib
* Seaborn
